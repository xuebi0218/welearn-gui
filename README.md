> 一款面向高校 WeLearn 在线学习平台的可视化辅助学习工具，基于图形界面实现课程学习自动化管理，全程本地运行，保障账号信息安全。

## ✨ 项目介绍
本项目基于 Python 开发，使用 PyQt5 构建可视化操作界面，封装了 WeLearn 平台开放接口，主要用于辅助完成线上课程视频学习任务，支持自动播放、进度同步、多任务队列管理，极大简化重复的人工操作流程。

核心设计特点：
- 纯本地运行，无第三方中转服务器，数据不对外泄露
- 图形化交互界面，零基础用户可直接上手使用
- 支持多课程批量任务处理，适配多门课程同步学习
- 学习进度实时与官网同步，数据保持一致

## 🎯 核心功能
1. **账号登录模块**
   - 支持学号/手机号+密码登录
   - 本地加密存储 Cookie，实现免重复登录
2. **课程信息加载**
   - 一键拉取个人所有已选线上课程
   - 区分已完成/未完成学习任务，直观展示进度
3. **视频自动化学习**
   - 自动识别未完成的视频学习任务
   - 适配平台校验规则，支持合理倍速设置
   - 异常状态自动重试，断网后可恢复继续学习
4. **任务队列管理**
   - 支持多课程批量添加至学习队列
   - 实时展示当前学习状态、进度详情
5. **自定义配置项**
   - 自定义播放间隔、失败重试次数
   - 支持代理配置，适配校园网网络环境
